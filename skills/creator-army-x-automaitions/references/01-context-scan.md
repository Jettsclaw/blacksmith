# Context Scan Protocol

Run this BEFORE asking the user any questions. The goal: never start from
scratch when you don't have to. The skill compounds — every interaction
makes the next one better.

## AI Ad Lab inputs to check FIRST

Before the 4-step scan below, check whether these production-skill outputs
already exist for this brand:

- **`/voc` HTML doc** for this product — already produced or scheduled?
  If yes, read it strategically per `16-skill-orchestration.md` Appendix A
  rather than re-doing customer excavation.
- **`/brand-dna` HTML doc** for this brand — already produced?
  If yes, read it per `16-skill-orchestration.md` Appendix B for hard
  visual + verbal constraints.
- **`/spy` swipe file** for this niche — recently run?
  If yes, you already have competitor winners to model or differentiate against.
- **Past `/create-static-ads` / `/multiplier` / `/rebuild` outputs** —
  what's been produced before? What got spent on?

If any of these exist, USE them. Don't re-research what's already documented.
If none exist, the scan below tells you whether to commission them.

## The Scan (Do All 4 in Order)

### 1. Memory + Conversation History

Search for this brand name, URL, founder name, or product names in:
- Your conversation history with this user
- Long-term memory / stored context
- Previous sessions or threads

**What you're looking for:**
- Previous ad briefs or creative concepts generated
- Customer types previously identified (barriers, motivators, personas)
- Performance data the user shared (what worked, what didn't)
- Reviews or customer language previously discussed
- Offer structures or pricing previously analyzed
- Competitor analysis previously done
- Brand voice or tone guidelines previously established

### 2. Workspace Files

If you have access to the user's file system or project directory, check for:
- Analytics reports, ad performance CSVs or dashboards
- CRM exports or customer data
- Previous creative briefs, scripts, or storyboards
- Brand guidelines or style guides
- Product data, pricing sheets, or catalog files
- Competitor research documents
- Review exports or NPS data

### 3. Connected Tools

If you have access to any of these, pull relevant data:
- **Meta Ads / Facebook Ads Manager** → recent performance by creative,
  which ads are getting spend vs not, hook rates, hold rates, CPA by ad
- **Shopify / Ecommerce platform** → top products by revenue, AOV,
  customer demographics, repeat purchase rates, product reviews
- **Google Analytics** → traffic sources, landing page performance,
  conversion funnels, audience demographics
- **CRM** → customer segments, lifecycle stages, common objections from
  sales notes, deal win/loss reasons
- **Review platforms** → actual customer language, common praise,
  common complaints, star distribution

### 4. Build on Past Work

**If this is the FIRST brief for this brand:**
→ Proceed to full customer excavation (Step 1)
→ Ask all 4 excavation questions

**If you've generated briefs for this brand before:**
→ DO NOT start from scratch
→ Present what you already know:
  "Last time we identified these customer types: [list them].
  We generated [X] ad concepts. Do you have any data on what
  performed? What should we keep, change, or add?"

→ Ask targeted questions only for what's MISSING or CHANGED:
  "Has anything shifted in your customer base since last time?"
  "Any new products, offers, or competitors I should know about?"
  "Which of the previous ad concepts ran? What did Meta spend on?"

**If the user shares performance data:**
→ Run the Learning Loop (references/09-learning-loop.md)
→ Use the insights to generate the NEXT batch of creative
→ This is where the skill compounds — each batch is smarter than the last

## What This Changes

| Scenario | Without Context Scan | With Context Scan |
|----------|---------------------|-------------------|
| First brief | Ask 4 questions | Same — ask 4 questions |
| Second brief | Ask same 4 questions again | "Here's what I know. What's changed?" |
| After running ads | Generate blind | "Meta spent on ads 3, 7, 11. Let me analyze why and build the next batch." |
| New product, same brand | Start from scratch | "Same customer types apply. Let me adapt the angles for this product." |
| Same product, new quarter | Generic refresh | "Your Q1 winners used question hooks + demo bodies. Let me evolve that for Q2." |

## The Compound Effect

Brief 1: Cold start. Full excavation. 12 diverse ads.
Brief 2: Warm start. "What worked?" New batch informed by data.
Brief 3: Hot start. Agent knows the winning patterns. Generates creative
that builds on proven strategy with genuine creative diversity.
Brief 10: The agent knows this brand's customers better than most employees.
Every brief takes 5 minutes and produces higher-quality creative than the last.

This is the moat. Templates don't compound. This skill does.
